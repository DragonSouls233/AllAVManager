## 第一阶段：后端基础设施（commit: def8b72）

### 1. 模块数据库管理器 — `app/db/module_db.py`

- 每个模块使用**独立 SQLite .db 文件**（chinese.db / uncensored.db / fc2.db / pornhub.db）
- 单例模式 `ModuleDatabase.get_instance(module_name)`，全局共享引擎
- SQLite PRAGMA 优化：WAL 模式、busy_timeout=60s、foreign_keys=ON、synchronous=NORMAL、cache_size=-64MB、temp_store=MEMORY
- `init_all()` 批量初始化 4 个模块数据库
- `close_all()` 统一释放连接

### 2. 数据模型 — `app/db/`

#### ChineseMovie / ChineseActor (`chinese_models.py`)
- **ChineseMovie**: code, title, folder_name, folder_based_actors (JSON), extracted_actor, studio, cover_url, poster_url, thumb_url, trailer_url, sample_images, release_date, duration, rating, plot, genre, tag, source, source_url, file_path, file_size, fingerprint, play_count, view_status, status, created_at, updated_at, scraped_at
- **ChineseActor**: name, alias, avatar_url, source (folder/scraper), studio, movie_count
- 国产模块独有：`folder_name` + `folder_based_actors`（从文件夹名识别的演员列表）

#### Fc2Movie (`fc2_models.py`)
- FC2 独有字段：`is_mosaic`（是否有码）、`seller_id`（卖家ID）
- 番号格式：FC2-123456 / FC2PPV-123456

#### UncensoredMovie (`uncensored_models.py`)
- 独有字段：`source_platform`（来源平台索引）、`series`
- 番号格式：HEYZO-1234 / 111111-111 / RED0123

#### PornhubMovie (`pornhub_models.py`)
- 独有字段：`source_id`、`source_views`、`source_score`、`source_downloads`、`uploader`、`categories`
- 番号格式：ph1234567890abcdef（viewkey）

### 3. 模块配置模型 — `app/config/module_models.py`

```python
class ModulesConfig:
    jav: ModuleConfig           # enabled, media_dirs
    uncensored: ModuleConfig

                                 #   folder_pattern, actor_blacklist,
                                 #   auto_create_actor, studio_names_as_folder
    pornhub: ModuleConfig
    scanner: ScannerConfig      # concurrent_limit, retry_count
```

### 4. 文件夹演员识别器 — `app/scraper/folder_actor.py`

国产模块核心功能，从文件夹名提取演员名：

- **黑名单过滤**：新建文件夹、合集、精选、unknown、未分类、tmp、temp、downloads 等
- **工作室名排除**：23 个已知工作室（麻豆传媒、天美传媒、果冻传媒、精东影业...）
- **多演员分割**：`+` / `.` / `_` / `&` / `，` / `、` / 空格
- **名称规则**：
  - 中文名：2-8 字（`[\u4e00-\u9fff]{2,8}`）
  - 英文名：首字母大写（`[A-Z][a-z]+`）
  - 混合名：中文+英文
- `extract_actor_from_folder()` / `clean_actor_name()` / `is_actor_folder()`

### 5. 扫描器基类 + 各模块扫描器 — `app/tasks/`

#### BaseScanner (`base_scanner.py`)
- 抽象基类，定义 `scan()` 接口
- `find_video_files()`: 递归查找视频文件
- `get_relative_path()`: 获取相对媒体目录的路径
- 支持扩展名：.mp4 .avi .mkv .mov .wmv .flv .webm

#### ChineseScanner (`chinese_scanner.py`)
- 文件夹深度可配置（1-5层）
- `generate_chinese_code()`: 生成唯一编码 `CN-{actors}-{hash8}`
- `extract_code_from_filename()`: 从文件名提取已知番号（MD/TM/91CM/MKY/SWAG/REAL/EDM/HKD/JVID）
- `extract_actor_from_filename()`: 从文件名提取中文名

#### Fc2Scanner (`fc2_scanner.py`)
- `extract_fc2_code()`: 支持 FC2-PPV-123456 / FC2-123456 / 纯数字6-7位 / [123456] 格式

#### UncensoredScanner (`uncensored_scanner.py`)
- `extract_uncensored_code()`: 支持 21 种无码前缀（HEYZO/1PONDO/CARIB/TOKYO-HOT/RED/PACOPACOMAMA...）
- 支持 `111111-111` 格式

#### PornhubScanner (`pornhub_scanner.py`)
- `extract_pornhub_code()`: 支持 `ph` + 10-20位十六进制 viewkey 格式
